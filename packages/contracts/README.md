# @trigora/contracts

Shared types for Trigora.

---

## Purpose

This package defines the core types used by Trigora:

- FlowDefinition
- FlowEvent
- FlowContext
- Trigger

---

## Usage

Most users should import from:

```ts
import { defineFlow } from '@trigora/sdk';
```

instead of using this package directly.

---

## Notes

This package is primarily used internally by the SDK and CLI.
